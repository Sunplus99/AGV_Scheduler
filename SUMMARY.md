## 🎓 面试准备

### 核心话术

**项目介绍**（30 秒）：
"我做了一个 AGV 智能调度系统，模拟仓储物流场景。系统采用 C++17 开发，使用 Reactor 网络模型和线程池实现高并发，支持 100+ AGV 同时在线。核心功能包括动态任务调度、A* 路径规划和实时避障。"

**技术深入**（1 分钟）：
"在网络层，我使用了主从 Reactor 模式，主 Reactor 负责 Accept，从 Reactor 负责 IO，充分利用多核 CPU。在调度层，我实现了贪心调度算法，并使用策略模式支持算法热插拔。在规划层，我使用 A* 算法进行路径规划，并通过内存池优化性能。在并发控制上，我使用了读写锁和无锁快照技术，保证了高并发下的正确性和效率。"

**亮点强调**（30 秒）：
"项目的亮点在于：一是高性能，通过 Epoll ET 模式和零拷贝优化，支持高并发；二是智能调度，通过无锁快照和双重检查，保证调度的正确性和效率；三是可扩展，通过策略模式和配置化设计，支持灵活扩展。"

## 🚀 快速演示

### 推荐配置（校招面试）

```bash
# 1. 修改配置文件
# - map.width/height: 50
# - map.ratio: 0.15
# - threads_num.io/worker: 4

# 2. 启动服务器（WARN 级别，减少日志）
./AgvServer ./config.json WARN

# 3. 启动 10 辆 AGV
./AgvSimulator 10

# 4. 观察日志
tail -f ./logs/agv_server.log
```

### 演示要点

1. **启动阶段**
   - 地图初始化
   - AGV 登录
   - 任务生成

2. **运行阶段**
   - 任务分配
   - 路径规划
   - AGV 移动

3. **完成阶段**
   - 任务完成统计
   - 性能指标
